## Troubleshooting

### Site shows old README content?
**Solution:** Wait 2-3 minutes for GitHub Pages to rebuild. Then refresh with Ctrl+Shift+R (hard refresh).

### Images not loading?
**Solution:** Check GitHub Actions for deployment status. All images are now in `docs/static/images/`.

### Want to update the dashboard?
**Solution:**
```bash
cd /home/arvind/Downloads/projects/Working/urban_heat_island

# Option 1: Edit in docs/ directly
nano docs/index.html

# Option 2: Edit in web_dashboard/ then copy
nano web_dashboard/index.html
cp web_dashboard/index.html docs/

# Then commit and push
git add docs/
git commit -m "Update dashboard: [your changes]"
git push origin main

# Wait 1-2 minutes for GitHub Pages to rebuild
```
